# Lantern Ruby Client

No Ruby client is required for `pg` or `Sequel`. For `ActiveRecord` and `Rails`, you can use the `lantern` gem.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'lantern'
```

And then execute:

```bash
bundle
```

Or install it yourself as:

```bash
gem install lantern
```

## Rails

```bash
rails generate lantern:install
rails db:migrate
```
